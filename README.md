# Capstone Project Module 2

## Customer Transaction and Delivery Performance Analysis on Tokopedia

### Overview

This project was completed as the Capstone Project for Module 2 of the Purwadhika Job Connector Data Science & Machine Learning Program.

The objective of this project is to analyze customer transaction patterns and delivery performance using Python and Tableau in order to generate business insights that support decision making.

---

## Business Objectives

- Analyze customer transaction characteristics.
- Analyze delivery performance.
- Generate business insights and actionable recommendations.

---

## Dataset

The project uses three datasets:

- users.csv
- transactions.csv
- deliveries.csv

---

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Tableau
